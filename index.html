Electron Configuration

The electron configuration of hydrogen is 1s¬π. This means it has one electron in the first energy level, specifically in the s orbital.

Hybridization Information

Key Properties

Hydrogen is the lightest element. Under standard conditions, hydrogen is a colorless, odorless, tasteless, non-toxic, nonmetallic, highly combustible gas. It's the most abundant chemical substance in the universe.

About PeriodicTable.io

PeriodicTable.io is an interactive web application designed to make learning about chemical elements engaging and accessible.

Features include:

  • Complete information on all 118 elements
  • Visual representations of atomic structure
  • Interactive hybridization models
  • Property comparisons
  • Practical applications for each element
  • Responsive design for all devices

This application is designed for students, educators, and anyone curious about chemistry and the elements that make up our world.

How to Use PeriodicTable.io

Basic Navigation

Click on any element in the periodic table to view detailed information about it.

View Options

Use the buttons above the table to switch between different views:

  • Standard View: Shows elements colored by their category
  • Electronegativity: Colors elements based on their electronegativity values
  • Atomic Radius: Colors elements based on their atomic radius
  • Ionization Energy: Colors elements based on their first ionization energy

Element Details

When you click on an element, you'll see various tabs with different information:

  • Atomic Structure: Visual representation of the element's electron configuration
  • Hybridization: Shows orbital hybridization patterns
  • Properties: Displays key physical and chemical properties
  • Applications: Lists common uses of the element

Mobile Usage Tips

On smaller screens, you can scroll the periodic table by dragging. Use the horizontal scroll on tabs if all tabs don't fit on screen.
